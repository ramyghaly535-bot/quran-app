# 🕌 قرآن في كل زمان ومكان

تطبيق ويب متكامل للاستماع إلى القرآن الكريم مع +80 قارئ

## ✨ المميزات

- 🎤 **+80 قارئ** من أشهر قرّاء القرآن الكريم
- 📖 **114 سورة** مع تفسير وترجمة آية بآية
- 🔊 **تحكم كامل بالصوت** (رفع/خفض)
- ❤️ **قائمة المفضلات** لحفظ القرّاء المفضلين
- 🔍 **بحث عن قرّاء** من الإنترنت وإضافتهم
- ⬇️ **تحميل السور** بصيغة MP3
- 📱 **تطبيق PWA** يعمل بدون إنترنت بعد التثبيت
- 📐 **تصميم متجاوب** يعمل على الجوال والكمبيوتر
- 🌙 **خلفيات طبيعية** متعددة مع تأثير الضبابية
- 🔄 **تكرار التلاوة** آية أو سورة كاملة
- ⏰ **عرض التاريخ** الهجري والميلادي
- 📲 **مشاركة عبر QR** وروابط مباشرة
- 💬 **تواصل عبر واتساب**

## 🛠️ المتطلبات

- **Node.js** 18 أو أحدث
- **Bun** (مُوصى به) أو npm

## 📦 التثبيت

```bash
# فك الضغط عن المشروع
tar -xzf quran-app-backup.tar.gz
cd quran-app

# تثبيت المكتبات
bun install
# أو: npm install

# تشغيل المشروع
bun run dev
# أو: npm run dev

# فتح في المتصفح
# http://localhost:3000
```

## 🚀 النشر

### Vercel (مجاني)
```bash
npm i -g vercel
vercel
```

### Netlify (مجاني)
```bash
npm run build
# رفع مجلد .next/static وملف .next/server إلى Netlify
```

### أي استضافة تدعم Node.js
```bash
bun run build
bun start
```

## 📁 هيكل المشروع

```
quran-app/
├── src/
│   └── app/
│       ├── page.tsx          # الصفحة الرئيسية (كل الكود)
│       ├── layout.tsx         # إعدادات HTML العامة
│       ├── globals.css        # جميع الأنماط
│       └── api/               # واجهات API
│           ├── ayah.ts        # آيات القرآن
│           ├── surah.ts       # بيانات السور
│           ├── audio.ts       # روابط الصوت
│           ├── translate.ts   # الترجمة
│           ├── tafseer.ts      # التفسير
│           └── download.ts    # التحميلات
├── public/
│   ├── manifest.json         # إعدادات PWA
│   ├── sw.js                 # Service Worker
│   ├── quran-logo.png        # الشعار (1024x1024)
│   ├── icon-*.png            # أيقونات بأحجام مختلفة

├── prisma/
│   └── schema.prisma         # قاعدة البيانات
├── package.json
├── next.config.ts
└── tsconfig.json
```
